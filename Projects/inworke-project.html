Freelance Inworke Website & Application

The Freelance Inworke platform is a school project designed to connect freelancers and clients seamlessly through a PHP-powered website and a React Native mobile application. By leveraging the power of a RESTful API, the platform ensures real-time synchronization and communication between the web and mobile interfaces. Freelancers can showcase their skills, bid on projects, and track their progress, while clients can post jobs, hire the best talent, and collaborate in real time. Key Features

  •  Real-Time Communication and Synchronization:  Changes made on one platform instantly reflect on the other, ensuring a consistent experience for users switching between mobile and web interfaces.
  •  Comprehensive Job Management:  Freelancers can bid on jobs, track ongoing projects, and manage their profiles, while clients can post jobs, review candidates, and hire the best match.
  •  Secure and Scalable System:  A robust backend built with PHP, secure RESTful API for data exchange, and scalable database design provide a safe and reliable platform for users.
