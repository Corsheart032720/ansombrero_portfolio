Pizza Ordering System

The Simple GUI Pizza Ordering System is a Java-based school project that provides a user-friendly platform for managing pizza orders. This system demonstrates the power of core Java in creating an intuitive graphical user interface (GUI) while streamlining the ordering process for various user roles, including Admin, Cashier, and Guest. The application allows users to log in, customize pizzas with various crusts and toppings, and track total sales efficiently. Designed with simplicity and functionality in mind, it ensures a seamless experience for users while offering robust features to handle transactions and manage sales data.

Technologies Used

Java

Key Features

  •  Access and Login System:  The application includes a secure and role-specific login feature. Admins and cashiers log in using predefined credentials, while guests can access the system without a login.
  •  Pizza Customization and Real-Time Sales Tracking:  Users can customize their pizzas by selecting crust types (Normal, Cheesy, Sausage) and adding extra toppings like Extra Cheese or Mushrooms, each with corresponding prices.
  •  Interactive GUI with Intuitive Navigation:  The application uses GUI components like JLabel, JComboBox, JCheckBox, JButton, and JTextField to create an engaging and easy-to-use interface. Users are guided through the process with clear prompts, error messages for incorrect login attempts, and confirmation dialogs for actions like logging out or resetting sales.
